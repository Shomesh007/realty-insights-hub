# Lyka Connect | LykaRealty | AI-Powered Dubai Real Estate Insights

Lyka Connect (LykaRealty) is a premium, AI-driven platform designed to provide institutional-grade real estate analytics for the Dubai market. It specifically caters to sophisticated investors by offering precision tools for ROI prediction, risk assessment, and wealth projection.

## Core Features

- **AI Income Analysis**: Predict rental yields and capital appreciation based on live market data.
- **Mortgage & EMI Strategy**: Advanced financial modeling for property financing.
- **Investment Risk Estimator**: Data-driven risk profiling for sustainable growth.
- **Wealth Projection**: Multi-year growth visualization for luxury portfolios.

## Technology Stack

- **Frontend**: React, TypeScript, Vite
- **UI & Styling**: shadcn/ui, Tailwind CSS
- **Icons**: Lucide React
- **Visualization**: Recharts
- **Backend API**: Azure OpenAI integration for AI insights

## Getting Started

### Local Development

1.  **Clone the repository**:
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    ```

3.  **Set up Environment Variables**:
    Create a `.env` file with the required API keys (refer to `.env.example` if available).

4.  **Start development server**:
    ```sh
    npm run dev
    ```

### Deployment

This project is configured for deployment on **Netlify**.

1.  Connect your repository to Netlify.
2.  Build settings:
    - **Build command**: `npm run build`
    - **Publish directory**: `dist`
3.  Ensure environment variables are configured in the Netlify dashboard.

## Deployment Config

- `netlify.toml`: Handles single-page application routing.
- `robots.txt`: Optimized for SEO and AI/LLM scrapers.
- `favicon.png`: Premium custom branding.
